# 📁 文件路径提取器 - 批量复制文件路径到剪贴板

> 简单易用的文件路径批量处理工具，让文件路径复制变得轻松高效！

## 🎯 功能介绍

这是一个专为普通用户设计的文件路径提取工具，能够帮助您：

- ✅ **批量提取文件路径** - 一次性处理多个文件路径
- ✅ **智能过滤重复项** - 自动去除重复的文件路径
- ✅ **自动排序整理** - 按文件名升序排列（不区分大小写）
- ✅ **一键复制到剪贴板** - 轻松复制到其他应用程序
- ✅ **多编码格式支持** - 自动识别 UTF-8、GBK 等编码
- ✅ **错误智能处理** - 遇到问题时自动尝试其他编码

## 🚀 快速开始

### 📋 使用前提

- Windows 7 或更高版本系统
- [不忙脚本盒子](https://www.bm-box.cn) - 用于快速调用脚本

### 📖 详细使用步骤

#### 通过不忙脚本盒子（推荐）

1. **安装不忙脚本盒子**
   
   - 访问 [官网](https://www.bm-box.cn) 下载安装
   - 完成软件安装和基本设置

2. **添加脚本到工具箱**
   
   - 通过脚本库将该脚本安装至不忙脚本盒子
   - 不忙脚本盒子会自动为脚本配置运行环境

3. **选择文件**
   
   - 在资源管理器中选择要处理的多个文件
   - 或在文本文件中准备好文件路径列表

4. **执行脚本**
   
   - 使用配置的快捷键快速调用
   - 或右键菜单选择"批量提取文件路径"

5. **获取结果**
   
   - 脚本自动提取和整理文件路径
   - 结果自动复制到剪贴板，可直接粘贴使用

### 编码支持

脚本自动支持以下编码格式：

- UTF-8（推荐）
- GBK/GB2312（Windows 中文系统）
- UTF-16

### 排序规则

- 默认按文件名升序排序
- 不区分大小写排序（abc.txt 和 ABC.txt 相邻排列）

### 过滤规则

- 自动移除空行和只包含空白字符的行
- 自动去除重复的文件路径
- 保持原始顺序的同时进行去重

## 📊 输出示例

假设您的输入文件包含以下内容：

```
C:\Users\张三\Documents\report.docx
C:\Users\张三\Pictures\photo.jpg
C:\Users\张三\Documents\report.docx
D:\工作\数据\分析.xlsx
```

脚本处理后输出的结果：

```
1. C:\Users\张三\Documents\report.docx
2. C:\Users\张三\Pictures\photo.jpg
3. D:\工作\数据\分析.xlsx
```

## ⚠️ 常见问题

### Q: 脚本无法运行怎么办？

A: 请确保：

1. 已安装 Python 3.6 或更高版本
2. 已安装不忙脚本盒子
3. 脚本文件路径正确

### Q: 复制到剪贴板失败？

A: 可能是以下原因：

1. 系统权限限制
2. 剪贴板被其他程序占用
3. 文件路径包含特殊字符

### Q: 支持哪些文件类型？

A: 脚本处理文本文件（.txt），支持各种文件路径格式。

### Q: 可以处理多少个文件路径？

A: 理论上没有限制，建议单次处理不超过10000个路径以确保最佳性能。
